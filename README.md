# SevenTinyBlog
Hexo 博客的源文件

## Hexo 环境搭建
1. Clone 当前仓库
2. 安装 Node.js
3. 安装 依赖项 `npm install`

## Hexo 命令
- 编译 `hexo g`
- 编译并部署 `hexo -g d`
- 本地Host `hexo server`
- 查看Hexo及插件版本 `hexo -v`

## 升级依赖包
1. 安装升级工具 `npm install -g npm-upgrade`
2. 执行升级工具 `npm-upgrade` 升级工具将自动修改 `package.json` 文件
3. 安装包 `npm update --save`
